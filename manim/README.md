## Manim setup
* Install `manim`: https://docs.manim.community/en/stable/installation.html
* VSCode extention: `Manim LiveView`
* Manim docs: https://docs.manim.community/en/stable/reference.html
Run via extention or generate with cli:
```commandLine
manim scene.py
```
